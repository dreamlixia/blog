### 开发运行此项目
---

### 全局安装vuepress
>```
>npm install -g vuepress
>```

### 在根目录安装node_modules
>```
>npm i
>```

### 修改文件：
>```
>可自行定义 src/xxx文件夹和src/xxx/xxx.md文件
>```

### 添加文件：
>```
>在src/xxx 下直接新建md文件即可
>```

### 添加图像：
>```
>请在与docs同级的images下添加，打包不会丢失
>```

### 运行：
>```
>npm run dev 或 vuepress dev docs
>```
