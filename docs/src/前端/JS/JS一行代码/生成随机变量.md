### 生成随机变量
---

生成随机颜色
---
```
const generateRandomHexColor = () =>
  `#${Math.floor(Math.random() * 0xffffff).toString(16)}`

// "#2042a2"
```

生成随机xxID
---
```js
// 生成用时间+随机简化生成 xxID 作为 TOS 文件名
Date.now()-Math.random().toString(36).slice(2)
// 得到一个类似于 '1721013040039-rdpisor5qa' 格式的随机串

// 或者使用 random方法，但没第一种干净好看
new Date().getTime()-Math.random()
// 得到一个类似于 '1721013388778-0.9659170465783229' 格式的随机串
