#### 可选链操作符?.
---

可选链操作符?.： 允许读取位于连接对象链深处的属性的值
```
var a = { a: 1 }
if (a && a.a && a.a.b) {
    console.log(111)
} else {
    console.log(222)
}
if (a?.a?.b) {
    console.log(111)
} else {
    console.log(222)
}
// 222
// 222
```

```
const player = {
    name: 'name',
    click: () => {
        return 'click'
    }
}

console.log(player?.name)// 'name'
console.log(player?.click?.())// 'click'
console.log(player?.passer)// undefined
```
